## 1. 系統介紹

### 1.1 系統簡介

企業管理系統是一套整合多項功能的內部管理平台，幫助您的團隊更有效率地協作。

### 1.2 主要功能

| 功能 | 說明 |
|------|------|
| 📋 任務管理 | 建立、指派、追蹤任務進度 |
| ✅ 每日任務 | 管理例行工作清單 |
| 📢 公告系統 | 發布重要訊息給團隊 |
| 💬 即時通訊 | 與同事即時溝通 |
| ⏰ 出勤打卡 | 上下班打卡記錄 |
| 📊 報表系統 | 建立和審核報表 |
| 💰 財務管理 | 記錄收支狀況 |
| 📄 SOP 文件 | 管理標準作業程序 |

### 1.3 角色權限

| 角色 | 可執行操作 |
|------|------------|
| **老闆 (BOSS)** | 全部功能、系統設定、人員管理 |
| **經理 (MANAGER)** | 部門管理、任務指派、報表審核 |
| **主管 (SUPERVISOR)** | 組員管理、任務追蹤 |
| **員工 (EMPLOYEE)** | 個人任務、打卡、聊天 |

## 2. 快速開始

### 2.1 首次設定（管理員）

1. 開啟系統網址
2. 看到「歡迎使用 企業管理系統」畫面
3. 設定管理員資料：
   - 輸入您的姓名
   - 設定登入帳號
   - 設定密碼
   - 上傳頭像（可選）
4. 點擊「建立帳號並開始使用」

### 2.2 登入系統

1. 開啟系統網址
2. 輸入帳號和密碼
3. 點擊「登入」

### 2.3 手機安裝（PWA）

**iPhone/iPad:**
1. 用 Safari 開啟系統網址
2. 點擊分享按鈕 (⬆️)
3. 選擇「加入主畫面」
4. 點擊「新增」

**Android:**
1. 用 Chrome 開啟系統網址
2. 點擊選單 (⋮)
3. 選擇「新增至主畫面」
4. 點擊「新增」

## 3. 功能操作說明

### 3.1 儀表板

登入後的首頁，顯示：
- 今日待辦任務
- 未讀公告數量
- 每日任務清單
- 目前時間與出勤狀態
- 打卡按鈕

### 3.2 任務管理

#### 查看任務
1. 點擊左側選單「任務管理」
2. 可依狀態篩選：待處理、進行中、已完成

#### 建立任務
1. 點擊「新增任務」按鈕
2. 填寫任務資訊：
   - 標題
   - 描述
   - 指派對象
   - 優先度
   - 截止日期
3. 點擊「儲存」

#### 更新任務狀態
1. 在任務卡片上點擊狀態按鈕
2. 選擇新狀態：進行中 / 已完成

### 3.3 每日任務

#### 查看今日任務
1. 在儀表板查看「每日任務清單」
2. 勾選已完成的項目

#### 管理每日任務模板（主管以上）
1. 點擊「部門文件與規範」
2. 選擇「每日任務」分頁
3. 新增/編輯任務模板

### 3.4 公告系統

#### 查看公告
1. 點擊左側選單「公告」
2. 點擊公告標題查看詳情
3. 閱讀後點擊「確認已讀」

#### 發布公告（主管以上）
1. 點擊「新增公告」
2. 填寫標題和內容
3. 選擇目標部門
4. 點擊「發布」

### 3.5 即時通訊

#### 開始聊天
1. 點擊左側選單「聊天」
2. 選擇聊天室或聯絡人
3. 在輸入框輸入訊息
4. 按 Enter 或點擊發送

#### 建立群組
1. 點擊「建立聊天室」
2. 輸入群組名稱
3. 選擇成員
4. 點擊「建立」

### 3.6 出勤打卡

#### 上班打卡
1. 在儀表板點擊「上班簽到」
2. 允許定位權限（首次）
3. 確認打卡成功

#### 下班打卡
1. 在儀表板點擊「下班簽退」
2. 確認打卡成功

#### 查看出勤記錄
1. 點擊「出勤管理」
2. 選擇日期範圍查看記錄

### 3.7 報表系統

#### 建立報表
1. 點擊「報表」
2. 點擊「建立報表」
3. 填寫報表內容
4. 點擊「提交」

#### 審核報表（主管以上）
1. 查看待審核報表
2. 點擊「審核」
3. 選擇通過或退回

### 3.8 財務管理（授權人員）

#### 新增收支記錄
1. 點擊「財務管理」
2. 點擊「新增記錄」
3. 選擇類型（收入/支出）
4. 填寫金額和說明
5. 點擊「儲存」

### 3.9 人員管理（管理員）

#### 新增員工
1. 點擊「人員管理」
2. 點擊「新增員工」
3. 填寫員工資料
4. 設定角色和部門
5. 點擊「儲存」

#### 編輯員工資料
1. 在員工列表點擊該員工
2. 修改資料
3. 點擊「儲存」

### 3.10 部門管理（管理員）

#### 新增部門
1. 點擊「人員管理」
2. 在部門區塊點擊「新增」
3. 輸入部門名稱
4. 選擇主題色和圖示
5. 點擊「儲存」

### 3.11 系統設定（管理員）

#### 資料備份
1. 點擊「系統設定」
2. 點擊「匯出資料」
3. 下載備份檔案

#### 系統重置
1. 點擊「系統設定」
2. 在危險操作區點擊「格式化系統」
3. 確認操作（此操作不可逆）

## 4. 常見問題

### Q1: 忘記密碼怎麼辦？
請聯繫系統管理員重設密碼。

### Q2: 無法登入，顯示「請等待 XX 分鐘」？
這是安全機制，連續輸入錯誤密碼 5 次會被暫時封鎖。請等待 30 分鐘後再試。

### Q3: 手機打卡定位失敗？
1. 確認已開啟定位權限
2. 確認 GPS 已開啟
3. 到戶外或窗邊嘗試

### Q4: 訊息沒有即時收到？
請重新整理頁面，或檢查網路連線。

### Q5: 如何修改個人資料？
1. 點擊左側個人頭像
2. 修改姓名或頭像
3. 點擊「儲存」

### Q6: 如何登出？
點擊左側選單底部的「登出」按鈕。
