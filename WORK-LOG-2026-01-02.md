## 🐛 解決的問題

### 1. 滾動問題
- **症狀**: 發送訊息時整個頁面被拖動
- **根因**: 使用 `scrollIntoView` 影響父容器
- **解決**: 改用 `scrollTop` 直接設置容器滾動位置

### 2. UI 更新問題
- **症狀**: 踢除成員後 UI 不更新
- **根因**: 成員列表從 `channel.participants` 計算，但該值未更新
- **解決**: 編輯模式下使用 `selectedMembers` 計算

### 3. 添加成員問題
- **症狀**: 踢除成員後無法重新添加
- **根因**: `availableUsers` 仍使用舊的 `channel.participants`
- **解決**: 同步使用 `selectedMembers` 判斷

### 4. 視覺反饋缺失
- **症狀**: 踢除成員按鈕無反應
- **根因**: 沒有 loading 狀態和視覺反饋
- **解決**: 添加 spinner、確認對話框、狀態變化
