## 🧪 API 測試命令

### 測試假期 API（使用 curl）

**1. 獲取 Token**
```bash
# 登入獲取 token
curl -X POST http://165.227.147.40:5000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"your_username","password":"your_password"}'
```

**2. 查看假期列表**
```bash
curl -X GET http://165.227.147.40:5000/api/leaves \
  -H "Authorization: Bearer YOUR_TOKEN"
```

**3. 創建假期申請**
```bash
curl -X POST http://165.227.147.40:5000/api/leaves \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "leave_type": "ANNUAL",
    "start_date": "2026-01-15",
    "end_date": "2026-01-17",
    "days": 3,
    "reason": "家庭事務"
  }'
```

**4. 批准假期**
```bash
curl -X POST http://165.227.147.40:5000/api/leaves/LEAVE_ID/approve \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "approval_notes": "批准",
    "conflict_override": false
  }'
```

**5. 駁回假期**
```bash
curl -X POST http://165.227.147.40:5000/api/leaves/LEAVE_ID/reject \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "approval_notes": "日期不適合"
  }'
```
