### 1. TypeScript 編譯錯誤修復

#### 修復項目：
1. **Task 介面缺少屬性** (`types.ts`)
   - 問題：`App.tsx` 使用了 `assignedToDepartment` 屬性，但 `Task` 介面中未定義
   - 解決：在 `types.ts:92-93` 添加 `assignedToDepartment?: string;` 屬性
   - 影響範圍：3 處錯誤 (App.tsx:374, 713, 721)

2. **MockApi 未定義** (`services/api.ts`)
   - 問題：檔案被截斷，`MockApi` 定義不完整導致編譯錯誤
   - 解決：在 `api.ts:604` 添加 `const MockApi = RealApi;`
   - 說明：由於 `USE_MOCK_API` 為 false，直接引用 `RealApi` 即可

3. **API 調用參數錯誤** (`App.tsx`)
   - 問題 1：使用 `is_archived` 而非 `isArchived` (line 424)
   - 問題 2：多個 API 方法傳入了多餘的 `currentUser` 參數
   - 解決：修復 6 處 API 調用錯誤
     - `tasks.updateProgress`: `is_archived` → `isArchived`
     - `users.create`: 移除第二個參數
     - `users.update`: 移除第二個參數
     - `users.delete`: 移除第二個參數
     - `departments.delete`: 移除第二個參數
     - `finance.delete`: 移除第二個參數
