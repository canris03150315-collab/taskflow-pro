# AI 重置指令快速參考

**創建日期**: 2026-01-24  
**用途**: 當對話太長或 AI 開始忘記規則時使用

---

## 🔄 何時需要重置？

### 觸發條件（任一符合）

- ✅ 對話超過 **20 輪**
- ✅ 對話超過 **30 分鐘**
- ✅ 執行了 **3 個以上**不同任務
- ✅ AI 開始**違反核心規則**
- ✅ AI 使用**過時資訊**

### 警告信號

如果 AI 出現以下行為，**立即重置**：

- ❌ 使用過時的版本號（不是從 WORK_LOG_CURRENT.md 獲取）
- ❌ 提到使用 PowerShell `&&`
- ❌ 建議跳過備份
- ❌ 使用錯誤的 Netlify Site ID
- ❌ 建議直接部署到生產（不經測試）
- ❌ 說「我記得」但給出錯誤資訊

## 🎯 使用時機

### 場景 1：長時間對話

**對話超過 20 輪或 30 分鐘時**：

```
🔄 我們已經對話了很久，請執行完整重置以確保規則正確。
```

### 場景 2：發現錯誤

**AI 給出錯誤資訊時**：

```
⚠️ 你剛才的資訊不正確，請重新閱讀 AI-MANDATORY-CHECKLIST.md 和 WORK_LOG_CURRENT.md。
```

### 場景 3：切換任務

**從一個任務切換到另一個任務時**：

```
🔄 我們要開始新任務了，請先執行快速重置確認系統狀態。
```

### 場景 4：部署前

**準備部署時**：

```
📋 部署前檢查：請確認當前版本資訊和部署流程。
```

## 💡 預防性提醒

### 建議頻率

- **每 10 輪對話**：要求 AI 自我確認核心規則
- **每個任務完成後**：快速版本確認
- **部署前**：完整規則檢查

AI 應該在以下時機主動提醒：

```

- 當前後端版本: [從 WORK_LOG_CURRENT.md 獲取]

- PowerShell 使用 ; 不是 &&
- 前端部署前必須清除 dist

是否需要我重新閱讀檢查清單？
```
