## 一、為什麼選擇 TaskFlow Pro

### 您的痛點，我們的解方

| 企業痛點 | TaskFlow Pro 的解決方案 |
|---------|------------------------|
| 任務分散在 Line、Excel、口頭交代 | **統一任務中心**，自動追蹤進度、逾期提醒 |
| 請假流程靠紙本或 Excel 傳來傳去 | **線上假勤系統**，主管一鍵審批、自動統計 |
| 公告訊息容易被員工忽略 | **企業公告欄**，已讀追蹤、重要通知強制顯示 |
| 績效考核靠主觀印象 | **KPI 自動計算**，任務完成率 + 出勤率客觀量化 |
| 財務流水帳難以彙整 | **零用金管理**，分類統計、自動生成報表 |
| 員工每天要開十幾個軟體 | **一站式平台**，所有功能整合在一個介面 |
| **多分公司資料難以整合** | **獨創總部聚合架構**，跨公司一鍵查詢 |

### 系統特色：獨創的「多公司分散部署 + 總部聚合」架構

大部分系統都是「一套系統裝所有客戶的資料」，當一家分公司出狀況時會影響全部。

**TaskFlow Pro 不一樣**：
- ✅ 每家分公司一台獨立伺服器、獨立資料庫
- ✅ 即使一家被攻擊或當機，其他分公司完全不受影響
- ✅ 總部仍可透過聚合層統一管理所有分公司
- ✅ 符合**資料隔離**與**風險分散**的法規要求

## 二、功能清單

### 核心功能（8 大模組）

#### 1. 任務管理
- 任務建立、指派、追蹤
- 優先級（低/中/高/緊急）
- 進度百分比 + 狀態管理
- 逾期自動標記 + 提醒
- 個人待辦、部門任務、全公司任務視圖

#### 2. 假勤系統
- 線上請假申請（特休、事假、病假...）
- 主管審批流程
- 出勤打卡 + 異常偵測
- 假單衝突自動檢查
- 請假統計報表

#### 3. 人事管理
- 員工基本資料
- 部門架構管理
- 四階角色權限（老闆/主管/組長/員工）
- 績效考核 KPI 自動計算
- 員工檔案查詢

#### 4. 財務管理
- 零用金登記
- 收入/支出分類
- 金額精度保護（不會因計算產生誤差）
- KOL 外部合作金流管理
- 財務報表匯出

#### 5. 公告中心
- 全公司公告
- 部門通告
- 緊急公告強制顯示
- 已讀追蹤（誰看過、誰沒看）

#### 6. 提案討論區
- 員工向上提案
- 留言討論
- 狀態追蹤（審核中/採納/駁回）

#### 7. 工作日誌
- 每日工作記錄
- 主管查閱員工進度
- 週報月報自動生成

#### 8. 備忘錄
- 個人便利貼
- 顏色分類
- 快速記事

### 🌟 AI 智能助理（獨家亮點）

這是 TaskFlow Pro 最大的差異化優勢。**員工只要用說的，就能操作系統**：

| 員工對 AI 說 | AI 自動執行 |
|------------|------------|
| 「幫我建一個明天要交的任務」 | 建立任務 + 截止日期 + 提醒 |
| 「我要請明天的事假」 | 自動填寫假單並送審 |
| 「記一筆 500 元支出」 | 建立財務紀錄 |
| 「分析這個月的工作狀況」 | 生成完整工作分析報表 |
| 「發公告提醒大家開會」 | 建立公告並推送 |

**安全保護**：
- 🛡️ 危險操作（刪除、批准）會要求二次確認
- 🛡️ AI 無法繞過使用者權限（員工不能透過 AI 做主管才能做的事）
- 🛡️ 防 Prompt Injection 攻擊（駭客無法用假指令騙 AI）

### 🌟 總部跨公司管理（集團客戶專屬）

如果您有多家分公司，總部可以：
- 一鍵查詢所有分公司的任務、請假、財務狀況
- AI 助手跨公司回答「所有分公司這週有幾件逾期任務？」
- 統一的儀表板顯示各分公司健康度
- 緊急狀況快速通知全集團
